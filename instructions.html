MRClicky Anleitungen

Erste Schritte

Nach der Installation von MRClicky können Sie Shortcuts für Ihren Mac erstellen und ausführen.

Shortcuts hinzufügen

1. Öffnen Sie MRClicky.
2. Tippen Sie auf „Neuen Shortcut hinzufügen“.
3. Geben Sie den AppleScript-Code ein oder wählen Sie eine Vorlage.
4. Speichern Sie den Shortcut für zukünftige Nutzung.

Shortcuts ausführen

1. Tippen Sie auf einen gespeicherten Shortcut.
2. Der Befehl wird an Ihren Mac gesendet und dort ausgeführt.

Pro-Version Funktionen

Mit MRClicky Pro können Sie geplante Automatisierungen einrichten.
